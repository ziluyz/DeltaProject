To build and install library:
----------------------------
g++ -Wall -Werror -std=c++14 -fpic -c deltaproject.cpp
g++ -shared -o libdelpro.so deltaproject.o
sudo cp libdelpro.so /usr/lib
sudo chmod 0755 /usr/lib/libdelpro.so
sudo ldconfig

To build simulations:
--------------------
g++ -Wall -Werror -std=c++14 -o simulations main.cpp -ldelpro
